Connect to Google Analytics GA4

Custom Query Fields

You can build custom queries to refine the data you access from the Google Analytics API. Navigate to the GA4 Query Explorer.

Select the desired Google Analytics Account and Property. In the 'Set parameters' section ONLY choose Metric and Dimension parameters. The date range, the advanced options, and the attribute and filter parameters are not necessary and can cause issues with API requests from Precog.

After selecting the Metrics and Dimensions you need to verify that the query request is valid. Click on the Make Request button and check the Response below. If there are issues you will be given an error message. Once you are satisfied with the query response select the 'Show request JSON' option next to the Make Request button. Copy the JSON query using the copy icon (right side of dialogue box) to ensure the entire query is copied. Copy each JSON query separately and paste in the Custom Query in the fields below.
